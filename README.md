# U Kostka Krasnobród

Strona internetowa dla obiektu noclegowego **U Kostka** w Krasnobrodzie.

Projekt został zbudowany jako statyczna strona w Astro. Zawiera podstrony z pokojami, galerią, plannerem pobytu, atrakcjami oraz kontaktem. Strona jest dostępna w wersji polskiej i angielskiej.

## Funkcje

- Strona główna z informacją o obiekcie
- Podstrona pokoi i obiektów
- Galeria zdjęć
- Planer pobytu z propozycjami atrakcji
- Resetowanie planu pobytu
- Pobieranie planu jako PDF przez drukowanie
- Lista atrakcji w Krasnobrodzie i okolicy
- Strona kontaktowa z telefonem, Facebookiem i mapą
- Wersja polska i angielska
- Responsywny layout

## Technologie

- Astro
- Tailwind CSS
- Alpine.js
- TypeScript

# U Kostka Krasnobród

Website for the **U Kostka** guest accommodation property in Krasnobród, Poland.

The project was built as a static website in Astro. It includes pages for rooms, gallery, stay planner, attractions and contact. The website is available in Polish and English.

## Features

- Homepage with information about the property
- Rooms and buildings page
- Photo gallery
- Stay planner with attraction suggestions
- Stay plan reset option
- Downloading the stay plan as a PDF through browser printing
- List of attractions in Krasnobród and nearby areas
- Contact page with phone number, Facebook link and map
- Polish and English versions
- Responsive layout

## Technologies

- Astro
- Tailwind CSS
- Alpine.js
- TypeScript

## Local Development

```bash
npm install
npm run dev
